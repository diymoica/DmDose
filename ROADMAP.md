### 📋 After v1.0.0 — What's planned

#### Core improvements
- [ ] **Additional module support** — connect multiple ESP32 from the same dashboard
- [ ] **100% customizable names and colors** from the dashboard (no theme editing needed)
- [ ] **Adaptive dashboard** — auto-adapts to the number of active pumps
- [ ] **Single-dose mode** — for one-shot injection systems
- [ ] **Weekly summary** — push notification every Monday with the week's report
- [ ] **Holiday mode** — scheduled pause for X days

#### New features
- [ ] **Dose confirmation** — wait for a flow sensor to validate the dose
- [ ] **Multi-system** — manage 2 separate installations from one HA instance
- [ ] **CSV export** — full dose history export

#### Hardware
- [ ] **ESP32 RTC guide** — keep time without WiFi (DS3231 module)
- [ ] **Magnetic stirrer** — dedicated companion ESP32 module
- [ ] **Hardware kit** — validated hardware list with purchase links

#### Améliorations du core
- [ ] **Support modules additionnels** — connecter plusieurs ESP32 depuis le même dashboard

- [ ] **Dashboard adaptatif** — s'adapte automatiquement au nombre de pompes actives
- [ ] **Mode dose unique** — pour les systèmes à injection ponctuelle
- [ ] **Résumé hebdomadaire** — notification push chaque lundi avec le bilan de la semaine
